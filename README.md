# Học Tiếng Anh với Kiko

Ứng dụng web học tiếng Anh vui vẻ, giống Duolingo, dành cho người Việt Nam.

## Tính năng
- Đăng nhập đơn giản
- Bài học tương tác với nhân vật
- Hình ảnh nhân vật và âm thanh feedback
- Theo dõi điểm số

## Cách chạy
1. Cài đặt dependencies: `npm install`
2. Chạy server: `npm run dev`
3. Mở http://localhost:5174/

## Thêm tài nguyên
- Thêm hình ảnh nhân vật vào `public/`: character1.png, character2.png, character3.png
- Thêm âm thanh vào `public/`: correct.mp3, wrong.mp3

Nếu không có file, app sẽ hiển thị text thay thế.
